Projetos do GitHub

  • Projeto em Arduino e C++ - Alarme De Incendio Inclusivo

    Sistema de alerta adaptado para ambientes como hospitais e clínicas, desenvolvido para atender pessoas com diferentes sensibilidades sensoriais, incluindo pacientes neurodivergentes, com TEA, esquizofrenia, ansiedade ou demência. O projeto utiliza LEDs com efeitos visuais suaves (fade) e buzzers com frequências ajustadas para evitar crises associadas a alarmes tradicionais. Implementado com Arduino, o sistema oferece uma abordagem mais inclusiva e segura para situações de emergência.

  • Projeto em Java - Sistema De Biblioteca

    Sistema de gerenciamento de biblioteca desenvolvido em Java, com foco em organização e controle de acervos. Implementa funcionalidades como cadastro de livros físicos e digitais, controle de empréstimos e devoluções, e armazenamento persistente de dados. Ideal para ambientes acadêmicos ou comunitários que necessitam de uma solução simples e eficiente para gestão de bibliotecas.

  • Projeto em Python - Alarme de Incendio com socket e client

    Sistema de alarme de incêndio acionado por sensor de chama ou botão, controlado por uma Raspberry Pi configurada como cliente em uma conexão socket. O cliente envia sinais via socket para um servidor em outra Raspberry Pi, acionando o alarme em resposta a eventos detectados. Este projeto demonstra a comunicação entre dispositivos em rede local, utilizando Python para o cliente e Arduino para o sensor de chama.
